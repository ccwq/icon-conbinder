# triage 标签

这些 skill 使用 5 个标准 triage 角色。本文件把这些角色映射到本仓库 issue tracker 里实际使用的标签字符串。

| mattpocock/skills 里的角色 | 本仓库里的标签 | 含义 |
| -------------------------- | -------------- | ---- |
| `needs-triage`             | `needs-triage` | 维护者需要先评估这个 issue |
| `needs-info`               | `needs-info`   | 等待提报者补充信息 |
| `ready-for-agent`          | `ready-for-agent` | 信息足够完整，可以交给离线 agent 处理 |
| `ready-for-human`          | `ready-for-human` | 需要人工实现 |
| `wontfix`                  | `wontfix`      | 不会处理 |

当 skill 提到某个角色时，使用上表对应的标签字符串。
